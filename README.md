# 巴西 Olist 电商平台口碑流失诊断与物流策略优化

## 项目背景
本项目深度挖掘了巴西最大集成电商平台 Olist 的 10 万余笔真实订单。通过全链路数据分析，量化了物流表现对品牌口碑的支配性影响，并基于统计学原理设计了“预期管理”优化实验。

核心目标：定位导致用户差评（1-2分）的结构性因素，并利用 A/B Test 验证改善方案。

---

## 核心洞察 

### 1. 物流时效是好评率的“生死线”
* 强相关：配送天数与差评率呈现线性正相关， r = 0.993。
* 边际效应：平均送达每延迟 1 天，差评率平均上升约 2%。
* 瓶颈月份：2018 年 3 月时效恶化导致评价崩盘（差评率达 22.8%）。

### 2. 品类风险的帕累托分布
* 结构性瓶颈：前 20% 的履约低效品类贡献了全平台近 68% 的物流负评。
* 办公家具特征：平均送达 21.1 天，平均得分仅 3.58。此类“大件重货”是平台满意度提升的核心阻碍。

### 3. 用户声量分析 
* 时效敏感：用户提及“物流时效”的频率是“产品质量”的 3.1 倍。

---

## 技术链路 

### 1. 数据建模 
使用 SQL 进行多表关联与特征提取，构建关键业务特征：
- Delivery Delta = 实际送达日期 - 预计送达日期
- Is Negative = 评分 <= 2 的二值化标签

### 2. 统计验证 
利用 Pandas 与 Scipy 进行 Pearson 相关性检验，锁定关键致偏因子。

### 3. 可视化 (Tableau)
搭建 3 层下钻仪表盘，涵盖：
- 核心指标看板 
- 时效-口碑追踪图 
- 品类风险矩阵 
---

## A/B 测试：预期管理实验设计
针对“大件物流难提速”的现状，提出预期管理策略：

* 实验组：算法预估 EDD 基础上主动预留 +2 天 缓冲期。
* 核心假设：缩短“预期与现实”的落差可有效提升用户满意度。
* 统计规格：
    * 显著性水平 alpha = 0.05 (Z_{alpha/2} = 1.96)
    * 统计功效 Power = 0.8
    * 护栏指标 下单转化率 (CVR)，设定红线跌幅为 0.5%。
